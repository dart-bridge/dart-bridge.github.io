dart2js

Before deploying your application, you must compile the client side Dart scripts into JavaScript using dart2js.

This is very simple:

= build

That's it. This command will compile and minify every .dart file in the public directory (most likely web/) into .dart.js files in the same location.

Your client side scripts should live in the lib/ directory. The scripts in web/ should only contain the main function, along with some initialization code, importing you main front end code base within the lib/ directory.

Note: As a default, .dart.js files will be ignored by Git. Either issue this command on your live server, deploy using another method than Git, or simply modify the .gitignore file in the root of the project.
