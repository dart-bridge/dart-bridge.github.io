Getting started

The recommended way to install Bridge is through the installer. It is available on pub and can be globally activated:

$ pub global activate new-bridge

Then, you can create new applications just by running the following command.

$ new-bridge my-blog
import 'dart:math' show Random;        // Import a class from a library. sdfg sdfg sdfg sdfg sdfg sdfg sdfg sdfg

class Die {                            // Define a class.
  static Random shaker = new Random(); // Define a class variable.
  int sides, value;                    // Define instance variables.

  String toString() => '$value';      // Define a method using shorthand syntax.

  Die({int n: 6}) {                   // Define a constructor.
    if (4 <= n && n <= 20) {
      sides = n;
    } else {
      throw new ArgumentError(/* */);  // Support for errors and exceptions.
    }
  }
  int roll() {                         // Define an instance method.
    return value = shaker.nextInt(sides) + 1; // Get a random number.
  }
}
